PHP ve Veritabanı İşlemleri

PHP ile MySQL veritabanı bağlantısı kurarak Veritabanı oluşturma, Veritabanına veri ekleme, sorgulama ve silme işlemleri yapabilirsiniz. Aşağıdaki adımlarda bu işlemleri nasıl gerçekleştirebileceğinizi öğrenebilirsiniz.

Adım 1: Veritabanına Gitme

İlk Önce veritabanı oluşturlmalıdır.XAMP üzerinden "Admin" butonuna tıklayarak phpMyadmin sayfasına gidilir.

Adım 2: Veritabanı Oluşturma

Sağ taraftaki "Yeni" sekmesine giderek veritabnına ad verilir ve "Oluştur" butonuna tıklayarak veritabanı oluşturlur

Adım 3: Veritabanına Bağlanma

PHP ile MySQL veritabanına bağlanmak için `mysqli_connect()` fonksiyonunu kullanabilirsiniz. Aşağıda bağlantıyı sağlamak için kullanılan örnek bir kod bulunmaktadır: 

<?php
        $servername = "localhost";
        $username = "root";
        $password = "";
        $dbname = "mydatabase";
        
        // Bağlantıyı oluştur
        $conn = mysqli_connect($servername, $username, $password, $dbname);
        
        // Bağlantı kontrolü
        if (!$conn) {
            die("Bağlantı başarısız: " . mysqli_connect_error());
        }
        ?>

Adım 4: Veritabanına Veri Ekleme

Veritabanına veri eklemek için `INSERT INTO` SQL komutunu kullanabilirsiniz. Aşağıdaki örnekte, kullanıcı adı ve e-posta bilgilerini veritabanına ekliyoruz: 

<?php
            $sql = "INSERT INTO users (name, email) VALUES ('Ahmet', 'ahmet@example.com')";
            
            if (mysqli_query($conn, $sql)) {
                echo "Yeni kayıt başarılı!";
            } else {
                echo "Hata: " . $sql . "<br>" . mysqli_error($conn);
            }
            ?>

Adım 5: Veritabanındaki Verileri Sorgulama

Veritabanındaki verileri sorgulamak için `SELECT` komutunu kullanabilirsiniz. Aşağıda, veritabanındaki tüm kullanıcıları listeleyen bir sorgu örneği yer almaktadır: 

<?php
            $sql = "SELECT id, name, email FROM users";
            $result = mysqli_query($conn, $sql);
            
            if (mysqli_num_rows($result) > 0) {
                while($row = mysqli_fetch_assoc($result)) {
                    echo "id: " . $row["id"]. " - Ad: " . $row["name"]. " - E-posta: " . $row["email"]. "<br>";
                }
            } else {
                echo "0 sonuç";
            }
            ?>

Adım 6: Veritabanındaki Veriyi Silme

Veritabanındaki bir veriyi silmek için `DELETE` SQL komutunu kullanabilirsiniz. Aşağıdaki örnekte, bir kullanıcının verisini silen bir PHP kodu yer almaktadır: 

<?php
            $sql = "DELETE FROM users WHERE id=1";
            
            if (mysqli_query($conn, $sql)) {
            echo "Kayıt silindi!";
            } else {
            echo "Hata: " . $sql . "<br>" . mysqli_error($conn);
            }
            ?>
